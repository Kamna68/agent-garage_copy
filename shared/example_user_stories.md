# Example User Stories for Testing

## Example 1: Simple Feature
As a user, I want to reset my password so that I can regain access to my account if I forget it.

## Example 2: E-commerce Feature
As a customer, I want to add items to my shopping cart so that I can purchase multiple products in a single transaction.

## Example 3: Admin Feature
As an administrator, I want to export user data to CSV so that I can analyze user statistics in Excel.

## Example 4: Security Feature
As a user, I want to enable two-factor authentication so that my account is more secure.

## Example 5: Search Feature
As a user, I want to filter search results by date and category so that I can find relevant content quickly.

## Example 6: Notification Feature
As a user, I want to receive email notifications for important updates so that I stay informed about my account activity.

## Example 7: API Feature
As a developer, I want to use API rate limiting so that the system remains stable under high load.

## Example 8: Mobile Feature
As a mobile user, I want the app to work offline so that I can access my saved content without internet.

## Example 9: Payment Feature
As a customer, I want to save my payment methods so that checkout is faster for future purchases.

## Example 10: Profile Feature
As a user, I want to update my profile picture so that other users can recognize me easily.
